Задание по Git:
1. Зарегистрировать аккаунт на Github.
2. Установить на свой компьютер Git.
3. Сгенерировать пару ssh ключей для подписания. Добавить публичный ключ в Github на своём аккаунте.
4. Сконфигурировать свои user.name и user.email в Git Bash.
5. Создать на Github репозиторий. Склонировать его на компьютер.
6. В выгруженном локальном репозитории (подсказка: это будет просто папка) создайте новый файл формата txt. Напишите туда любой текст (желательно цензурный) и сохраните изменения.
7. Сделайте коммит с любым комментарием.
Отправьте его на сервер (подсказка: через git push).
8. Проверьте, что файл появился на Github в репозитории, убедитесь, что комментарий отображается у коммита.
8* (необязательно) Откройте файл в Github, добавьте какой-либо текст. Сделайте коммит, не забыв указать комментарий. После этого в локальном репозитории (через Git Bash) сделайте pull и убедитесь что коммит, который вы создали на github, подтянулся, и текст в вашем файле на компьютере дополнился изменениями.
9. Создайте ветку feature и добавьте в неё два коммита.
10. Смержите ветку feature в main.
11*. (необязательно) Просмотрите, как выглядит ваша история коммитов, используя команду gitk&, поработайте с коммитами в Git GUI. Оцените удобство интерфейса.
